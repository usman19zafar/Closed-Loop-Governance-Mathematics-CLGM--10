Usable model classes (indestructible set):

Partially Observable Markov Decision Process (POMDP)
Handles hidden state + decisions under uncertainty
→ extends state-space + control to open systems
Robust / Adaptive Control
Uncertain dynamics but bounded
→ guarantees: system stays inside safety envelope despite drift
Stochastic Hybrid Systems
Discrete (policies) + continuous (metrics) + randomness
→ closest to real governance systems
Dynamic Bayesian / Causal Models
State evolution + intervention logic
→ moves from correlation → controllable causation
Game-Theoretic Models (Multi-Agent)
Agents with competing objectives
→ needed for organizational + AI interaction safety
Runtime Verification + Temporal Logic
Properties checked continuously (not just proven once)
→ invariants enforced during execution
Constrained Reinforcement Learning (Safe RL)
Learning + constraints simultaneously
→ adapts while respecting safety limits
